# Les Routes qui Sauvent des Vies

## Objectif

Développer un système de routage intelligent pour les véhicules de secours (ambulances, pompiers) capable de :

- **Choisir** le meilleur chemin possible avec les informations disponibles
- **S'adapter** en temps réel quand les conditions changent
- **Expliquer** chaque décision prise pour assurer la traçabilité

## Le Défi

Dans une grande ville africaine de 2 millions d'habitants, les ambulances mettent en moyenne **23 minutes** à arriver sur place après un appel aux secours. Le problème n'est pas le manque de véhicules, mais l'incertitude sur le terrain :

- Routes de natures variables (goudronnées vs pistes latérites)
- Trafic imprévisible selon l'heure et les incidents
- Informations incomplètes ou retardées sur l'état du réseau
- Conditions changeantes (pluie, inondations, accidents)

## Approche Requise

Ne pas résoudre le problème classique du plus court chemin, mais un problème de **décision dans l'incertitude** où le système doit naviguer avec des données imparfaites et réagir instantanément aux changements.
